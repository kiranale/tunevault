FMW Oracle_Home (WLS)
?
The WebLogic Server Oracle Home (e.g. /u01/install/APPS/fs1/FMW_Home/Oracle_Home). Contains WLS 12.2.1.x binaries — the application server hosting EBS. Patches here cover WLS PSUs and JVM updates. Must be patched to match oracle_common. Under online patching (ADOP), both fs1 and fs2 have separate FMW homes.
⚙️
Oracle_Home WLS
WebLogic Server binaries — WLS PSUs and JVM patches
The Oracle_Home under $FMW_HOME is the WebLogic Server Oracle Home. Patches here cover WLS PSUs (Patch Set Updates) and JVM security fixes. Silent failure mode: a patch applied to oracle_common but missing here (or vice versa) causes JRF dependency mismatches that can prevent WLS from starting after an ADOP cutover.
$FMW_HOME/Oracle_Home/OPatch/opatch lsinventory -oh $FMW_HOME/Oracle_Home
Output
webtier (OHS)
?
